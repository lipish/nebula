# Nebula i18n 验收清单（默认中文 + 中英切换）

## 0) 前置条件
- 前端可访问：`http://10.21.11.92:5173/`
- BFF 可访问：`http://10.21.11.92:18090/api/healthz`
- 测试账号：`admin / admin123`
- 浏览器建议：Chrome（无插件干扰）

## 1) 登录页验收
- 打开登录页，确认默认显示中文文案。
- 输入账号密码登录，确认登录成功。
- 检查以下中文是否正常：标题、副标题、邮箱/密码、记住我、登录按钮、底部提示。
- 切换语言到 English，刷新页面后仍保持 English。
- English 下上述文案全部切换为英文，且不出现 i18n key（如 `login.xxx`）。

## 2) 全局语言与持久化
- 登录后点击右上角账户菜单，使用语言切换（中文/English）。
- 切换后当前页面文案应即时变化。
- 切换后刷新浏览器，语言保持不丢失。
- 退出登录再登录，语言仍保持上次选择（localStorage 持久化）。

## 3) 导航与壳层验收
- 侧边栏分组标题与菜单项：工作台/基础设施/资源/系统均应可双语切换。
- 顶部搜索占位符、账户菜单项（编辑资料、账户设置、退出登录）可双语切换。
- 任意页面切换后，语言状态一致。

## 4) 页面级抽样（核心）
按“中文 -> English -> 中文”切换各检查一次：

### Dashboard
- 卡片标题、图表标题、图例、空态文案均正确切换。
- 表格头部与空态提示无硬编码残留。

### Model Service
- 页标题/副标题、筛选状态、表头、按钮（Start/Stop/Details/Delete）可双语切换。
- 复制模型名失败提示可双语切换。

### Load Model Dialog
- 三个步骤标题与描述（搜索/模板/配置）全部切换。
- 搜索占位、热门模型、手动输入提示、模板加载/空态、部署中按钮文案正常。
- 硬件区域提示、占用/卸载、多 GPU 提示、引擎与模型配置字段全部切换。

### Inference
- 过载告警、统计卡片、路由表头、引擎卡片、图表标题、端点表头全部切换。
- 空态不出现英文硬编码或 key。

### Model Catalog / Library
- 目录搜索区、下载任务条、下载弹窗、表头、操作按钮均切换。
- 模型库表头、状态、迁移路径弹窗文案均切换。

### Templates / Images
- 模板页表头、部署弹窗字段与按钮文案切换。
- 镜像页注册/编辑弹窗的 Label、Placeholder、策略选项、按钮切换。

### Endpoints / Nodes / Audit / Settings / Profile / Account
- 页标题、副标题、表头、筛选项、按钮、提示语都可切换。
- 管理员用户管理区（用户名/密码/角色）可切换。

## 5) 质量门槛（通过标准）
- 不出现 `xxx.yyy` 这类翻译 key 直接展示。
- 不出现明显混用（中文页面夹英文、英文页面夹中文）
- 刷新后语言不丢失。
- 页面功能不受影响（登录、加载模型、模板部署、镜像编辑等）。

## 6) 快速问题定位
若发现异常，优先检查：
1. `frontend/src/lib/i18n.tsx` 是否存在对应 key（zh/en）。
2. 目标页面是否使用 `t('...')`，是否仍有硬编码字符串。
3. 切换逻辑是否在 `App` 账户菜单生效，默认语言是否为 `zh`。
